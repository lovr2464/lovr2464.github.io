c++ STL常用数据结构

常用STL数据结构的使用

本人对c++不是很熟悉，之前使用STL时也遇到了些构造函数或者可用方法不了解的问题，在此进行一下整理

1. vector

构造函数

  • vector<T>(): 默认构造函数
  • vector<T>(size_t n): 创建包含 n 个元素的 vector
  • vector<T>(size_t n, const T& val): 创建包含 n 个元素，每个元素初始化为 val 的 vector
  • vector<T>(const vector<T>& other): 拷贝构造函数
  • vector<T>(vector<T>&& other): 移动构造函数
  • vector<T>(initializer_list<T> il): 初始化列表构造函数

常用方法

  • size(): 返回元素个数
  • empty(): 检查容器是否为空
  • push_back(const T& val): 在末尾添加元素
  • pop_back(): 删除末尾元素
  • front(): 返回第一个元素
  • back(): 返回最后一个元素
  • operator[](size_t idx): 访问指定位置的元素
  • at(size_t idx): 访问指定位置的元素，带范围检查
  • clear(): 清空所有元素
  • insert(iterator pos, const T& val): 在指定位置插入元素
  • erase(iterator pos): 删除指定位置的元素
